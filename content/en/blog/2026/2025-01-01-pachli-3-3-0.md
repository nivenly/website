This release:

- Allows you to switch between writing posts and browsing timelines using Android’s “Recents” screen.
- Adjusts the UI when writing a post because of the new quote-related controls.
- Has a number of bug fixes.

See the [full Pachli 3.3.0 release post](https://pachli.app/pachli/2025/12/30/3.3.0-release.html?utm_medium=referral&utm_source=nivenly.org&utm_campaign=release-3.3.0) for more details on these and other changes.
